## 等宽字体

等宽字体（Monospaced Font）是指字符宽度相同的电脑字体。与此相对，字符宽度不尽相同的电脑字体称为比例字体。

等宽字体适合编程，比例字体可以提高单词的可读性。

```css
.mono {
	font-family: Monaco, Courier, Monospace;
}
```
