# 音乐 - 古曲

广陵散相传是东汉末年流行于广陵（今江苏扬州）的民间乐曲，当时用琴、筝、笙、筑等乐器演奏。汉代应璩与刘孔才的信说“听广陵之清散”。广陵散描写战国时代铸剑工匠之子聂政为报严仲子知遇之恩，刺死韩相侠累，然后自杀的故事，因此通常认为是“聂政刺韩相”的别名。
