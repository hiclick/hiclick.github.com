# Book - 人文·历史

历史，或简称史，指人类社会过去的事件和行动，以及对这些事件行为有系统的记录、诠释和研究。历史可提供今人理解过去，作为未来行事的参考依据，与伦
理、哲学和艺术同属人类精神文明的重要成果。

历史的第二个含义，即对过去事件的记录和研究，又称为“历史学”，或简称“史学”。
