奥迪A3（Audi A3）是一款由奥迪生产的小型汽车，有两厢车和三厢车两种车型。它的主要竞争对手包括梅赛德斯-奔驰A级、宝马1系等。

第三代是于2012年推出，除了有三门、五门掀背外，也有房车版本。

2014年3月21日，奥迪A3作为一汽-大众汽车有限公司在华推出的第五款国产车型于上海正式上市发售，涵盖了国产两厢、进口两厢以及进口三厢在内的多款车型。
