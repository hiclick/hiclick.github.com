#### Give up CLA 220 4MATIC

以下前两节文字，来自车家论坛。

> 买奔驰任何一款车都不会有错；
>
> 想看十年后的汽车配置就看现款奔驰！

----------

能探测前方车辆急剧减速的情况下先亮警示灯再报警，听到报警后就一定要踩刹车了，要不一定会撞上了。

奔驰只要挂Ｒ挡，右后视镜自动调整压下收拢，能让我清楚看到车后的情况，挂Ｄ时前行几十米自动恢复正常。

主动刹车报警这个我也体会过了 非常实用 也救了我的车和我一次 那次我和副驾的妹子聊天没注意前面的急刹 我的220神奇的自动开始刹车了并且警报音响我才反应过来急刹 
就差一点撞上了 这个系统真的很好 很赞。

其实什么车不重要，不出事才最重要。以前我开个十万块的车都想跟别人比速度，后来用车屁股甩倒一对电单车男女后，我才深夜体会到，什么车在事故面前都非常的脆弱。
那次是因为前面的电单车一直不让行，我插到他前面，一转方向，尾巴就挤到他了，我在反光镜上看到他是慢慢失去平衡到在地上，不是很严重，我没理他油门踩到底，跑了，
后来我停好车再骑个单车去看看他死了没有，没人围观，应该是没事走人了。那次后我告诫自己，不管开什么车，都不要再有任何傲气，不要看不起任何开车骑车的，
不要再跟任何车较劲，所以到现在我都没有出个险，也没有因为骄傲发生险情。所以我对车的动力什么的不是特别追求，也相信车企敢做哪里有不够用的道理，
只是有些爱玩车的会喷什么车怎么怎么肉的，想想我真的没必要！

----------
